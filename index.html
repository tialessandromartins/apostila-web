📖 Missão: Uilizar ID e Class no Site de Receitas

Bem-vinda! Siga os passos abaixo para construir seu site. Marque ✅ quando terminar cada tarefa.

Etapa 1: Organizando as Pastas

Antes de digitar, precisamos de uma "casa" para o nosso projeto:

  • Vá até a Área de Trabalho do seu computador.
  • Clique com o botão direito, escolha Novo > Pasta.
  • Coloque o nome da pasta como: Meu_Site_Receitas

Etapa 2: O Conteúdo (HTML)

Abra o VS Code, vá em Arquivo > Abrir Pasta e selecione a pasta que você criou. Depois, crie o arquivo index.html e cole o código:

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Meu Caderno de Receitas</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>

    <h1 id="titulo-principal">Culinária da Vovó Maria</h1>

    <div class="cartao-receita">
        <h2>Bolo de Fubá Fofinho</h2>
        
        <h3 class="secao-titulo">Ingredientes</h3>
        <ul>
            <li class="item-lista">3 ovos inteiros</li>
            <li class="item-lista">2 xícaras de fubá</li>
            <li class="item-lista">1 colher de fermento</li>
        </ul>

        <h3 class="secao-titulo">Modo de Preparo</h3>
        <p class="texto-instrucao">Bata tudo no liquidificador, exceto o fermento. Depois, misture o fermento levemente e leve ao forno médio por 40 minutos.</p>
    </div>

    <hr>
    <p class="rodape">Criado com carinho na aula de Informática - 2026</p>

</body>
</html>

Etapa 3: O Visual (CSS)

Dentro da mesma pasta, crie o arquivo style.css e cole este código:

body { background-color: #fff9f0; }
#titulo-principal { color: #8b4513; text-align: center; }
.cartao-receita { 
    background: white; 
    border: 2px solid #deb887; 
    padding: 20px; 
    border-radius: 10px; 
}
.secao-titulo { color: #d2691e; }

Etapa 4: Resultado Esperado

Se tudo estiver correto, seu site ficará assim:

Culinária da Vovó Maria

Bolo de Fubá Fofinho

Ingredientes

  • 3 ovos inteiros
  • 2 xícaras de fubá
  • 1 colher de fermento

Modo de Preparo

Bata tudo no liquidificador, exceto o fermento e leve ao forno médio.
