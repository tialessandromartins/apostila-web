📖 Missão: Utilizar ID e Class no Site de Receitas

Etapa 1: Organizando as Pastas

Crie uma pasta chamada Meu_Site_Receitas na sua Área de Trabalho.

Etapa 2: O Conteúdo (HTML)

No VS Code, crie o index.html e cole o código abaixo:

<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1 id="titulo-site">Culinária da Vovó Maria</h1>
    <div class="cartao">
        <div class="faixa-azul">Bolo de Fubá Fofinho</div>
        <h3>Ingredientes</h3>
        <ul>
            <li>3 ovos inteiros</li>
            <li>2 xícaras de fubá</li>
            <li>1 colher de fermento</li>
        </ul>
    </div>
</body>
</html>

Etapa 3: O Visual (CSS)

Crie o arquivo style.css e cole o código:

body { background-color: #fff9f0; text-align: center; }
#titulo-site { color: #5d4037; border-bottom: 1px solid #ddd; }
.cartao { 
    background: white; border: 1px solid #deb887; 
    border-radius: 15px; margin: auto; max-width: 500px; 
}
.faixa-azul { 
    background: #335999; color: white; 
    padding: 15px; border-radius: 5px; 
}
ul { display: inline-block; text-align: left; }

Etapa 4: Resultado Esperado

Culinária da Vovó Maria

Bolo de Fubá Fofinho

Ingredientes

  • 3 ovos inteiros
  • 2 xícaras de fubá
  • 1 colher de fermento

Modo de Preparo

Bata tudo no liquidificador e leve ao forno médio.
